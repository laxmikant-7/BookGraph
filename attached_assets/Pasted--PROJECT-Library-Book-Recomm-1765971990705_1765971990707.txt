/*==============================================================
 CORE CONCEPT
==============================================================*/
PROJECT_NAME = "Library Book Recommendation System"

CORE_IDEA:
- Books are represented as GRAPH NODES
- Relationships between books are GRAPH EDGES
- Recommendations are generated via GRAPH TRAVERSAL
- This is NOT a simple CRUD project

/*==============================================================
 DATA STRUCTURE REQUIREMENTS
==============================================================*/
PRIMARY_DATA_STRUCTURE:
- Graph (Adjacency List)

GRAPH_DETAILS:
- Node  -> Book
- Edge  -> Relationship between books
- Edge Weight -> Relationship strength

RELATIONSHIP_TYPES:
- Same Genre
- Same Author
- Similar Keywords
- Borrowed Together (simulated data)

/*==============================================================
 ALGORITHMS
==============================================================*/
TRAVERSAL_ALGORITHM:
- Breadth First Search (BFS)
- Depth limit = 2

RECOMMENDATION_LOGIC:
- Traverse graph starting from searched book
- Score related books based on:
    + Genre match
    + Author match
    + Borrow frequency
- Rank books using Priority Queue (Max Heap)

/*==============================================================
 SUPPORTING DATA STRUCTURES
==============================================================*/
SUPPORTING_DS:
- HashMap -> BookID → Book (O(1) lookup)
- Priority Queue -> Rank recommendations
- Stack (optional) -> Recently viewed books
- Queue (optional) -> Reading list

/*==============================================================
 FUNCTIONAL REQUIREMENTS
==============================================================*/
FEATURES:
1. Add new book
2. Delete book
3. View all books
4. Search book
5. Recommend top 5 related books

BOOK_ATTRIBUTES:
- Book ID
- Title
- Author
- Genre
- Keywords

/*==============================================================
 APPLICATION WORKFLOW
==============================================================*/
WORKFLOW:
1. Load books into Graph
2. Create relationships (edges)
3. User searches a book
4. Find book via HashMap
5. Run BFS traversal
6. Calculate recommendation score
7. Rank using Priority Queue
8. Display recommendations
